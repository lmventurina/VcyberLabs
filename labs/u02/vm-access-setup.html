VM & Networking Concepts

Virtual Machine (VM)

A software computer that, like a physical computer, runs an operating system and applications.

IP Address

A unique string of numbers separated by periods that identifies each computer using the Internet Protocol to communicate over a network.

Ping

A computer network administration software utility used to test the reachability of a host on an Internet Protocol (IP) network.

Firewall

A network security device that monitors and filters incoming and outgoing network traffic based on an organization's previously established security policies.
