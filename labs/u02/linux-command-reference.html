Linux Quick Ref

grep

Search for a specific string or pattern of text within a file.

sudo

"SuperUser DO". Allows a permitted user to execute a command as the superuser (admin).

Redirection (>)

Directs the output of a command into a file, overwriting existing content.

Append (>>)

Directs the output of a command into a file, adding to the end of existing content.
