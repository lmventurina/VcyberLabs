Vulnerability Terms

MBSA

Microsoft Baseline Security Analyzer. A tool to determine security state by assessing missing security updates and less-secure security settings.

Vulnerability

A weakness which can be exploited by a threat actor, such as an attacker, to perform unauthorized actions within a computer system.

Patch Management

The process of distributing and applying updates to software to correct errors (bugs) or security vulnerabilities.

Administrative Shares

Hidden network shares created by the Windows NT family of operating systems that allow system administrators to have remote access.
