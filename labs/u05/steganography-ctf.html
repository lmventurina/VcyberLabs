Stego Terms

Steganography

The practice of concealing a file, message, image, or video within another file, message, image, or video.

ExifTool

A platform-independent Perl library plus a command-line application for reading, writing and editing meta information in a wide variety of files.

Binwalk

A tool for analyzing, reverse engineering, and extracting firmware images.

Steghide

A steganography program that is able to hide data in various kinds of image- and audio-files.

CYBER.ORG Range Worksheet

Objective

Apply steganography tools (Okteta, Exiftool, Binwalk, Steghide) to solve challenges and find hidden flags.

Instructions:

Challenges

1. Text hidden in picture (CSA_ForL2C1.png)

Open in ImageViewer. Zoom in on top right quadrant.

2. Text hidden in bits (CSA_ForL2C2-05.jpg)

Tool: Okteta. Open file with Okteta. Check right column (ASCII).

3. Metadata (Pico_SoMeta.png)

Tool: Exiftool. Command: exiftool Pico_SoMeta.png

4. File nested in a file (GT_Reservoir2.JPG)

Tool: Binwalk. Command: binwalk -e GT_Reservoir2.JPG
Check extracted folder. Use w3m flag.jpg to view result.

Paste screenshot of your terminal showing the Binwalk extraction or w3m output

5. Steganography Software (Pico_WhatLies.jpg)

Tool: Steghide. Command: steghide --extract -sf Pico_WhatLies.jpg
Passphrase: key123. Read output: cat flagz

6. Apply skills: Hiding a file

Use man steghide to learn how to hide a text file in an image.
