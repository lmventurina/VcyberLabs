Basic SQL Commands

CYBER.ORG Range Worksheet

Objective

Understand the structure of a SQL database and the basic SQLITE commands used to manipulate data.

Instructions:

Part 1: Basic Queries

1. Start a Terminal.
2. Open the database: sqlite3 /home/kali/CourseFiles/IntroCybersecurity/chinook.db
3. Identify tables: .tables
4. Identify table structure: .schema albums

Query Practice

Format: SELECT column FROM table; (Always end with a ;)

5. Simple Select:
SELECT EmployeeID, LastName FROM employees;
SELECT * FROM albums;

6. Sorting (ORDER BY):
SELECT * FROM artists ORDER BY Name ASC;
SELECT TrackID, Name, Composer FROM tracks ORDER BY composer ASC;

7. Conditionals (WHERE):
SELECT Name, UnitPrice FROM tracks WHERE UnitPrice > 1;
SELECT FirstName, LastName FROM customers WHERE State = 'CA';

8. Advanced Logic (AND/OR/LIKE):
SELECT CustomerID, BillingCountry, Total FROM invoices WHERE BillingCountry != 'USA' AND Total > 20;
SELECT title FROM albums WHERE Title LIKE '%the%' OR Title LIKE '%of%';
