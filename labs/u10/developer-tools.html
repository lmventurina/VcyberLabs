DevTools Terms

Developer Tools

A set of web developer tools built directly into the browser, allowing you to inspect specific elements, debug JavaScript and CSS, etc.

DOM

Document Object Model. The data representation of the objects that comprise the structure and content of a document on the web.

Console

A tool used to log information, run JavaScript, and interact with the web page.

Network Tab

Allows you to monitor resources that are being uploaded or downloaded over the network in real time.

Part 3: Network Tab - Investigate GET method vs POST method

1. GET METHOD: Use the Developer Tools (shift + ctrl + I) to look closely at the login data transfer.
- Click anywhere in Gruyere and select "Inspect." Position the screen windows so that you can see the webpage on the left and the Inspect window on the right. In the Inspect window, click on the "Network Tab"
2. In the upper right of Gruyere, click "Sign in" and login with username: cheddar password: orange. Notice that the URL now includes the username and password!
3. In the Inspect window, click the "login?uid" item to inspect the details for this network activity. A new box will open on the right with the Headers tab open. Here we can see that the login used the GET method to send info. The username & password are also visible here.
