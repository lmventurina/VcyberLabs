Recon Terms

WHOIS

A query and response protocol that is used for querying databases that store the registered users or assignees of an Internet resource.

Registrar

An organization that manages the reservation of Internet domain names.

Top-Level Domain (TLD)

The last segment of the domain name (e.g., .com, .org, .edu).

OSINT

Open Source Intelligence. Data collected from publicly available sources to be used in an intelligence context.

Recon with WHOIS

CYBER.ORG Range Worksheet

Objective

Students will practice performing reconnaissance using the WHOIS tool.

Note: Your group will share one computer to perform this activity. This exercise is done in groups because if there are too many queries coming from one source (your school), then databases may temporarily block queries.

Instructions:

Domain Reconnaissance

You will do recon on these 7 domain names:
Aglaw.us, Aptera.us, Evolo.us, Quiescent.us, Replaygames.us, Wctf.us, Zonix.us

For each domain:
1. Visit the domain website to quickly determine the type of content.
2. Do a Whois lookup to gather the name and address of the Registrant Contact.
3. Look up the address with Google Maps/Streetview.

Closing Analysis

A target is most vulnerable to an attack (like swatting) when the attacker can find lots of recon information. In this exercise there are significant differences in the amount of information found for each domain simply by looking at the web site, a Whois lookup and Google maps.
