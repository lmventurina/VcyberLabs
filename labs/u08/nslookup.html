Using nslookup

CYBER.ORG Range Worksheet

Objective

Students will practice performing reconnaissance using the nslookup tool.
Background:
DNS (Domain Name Server) translates domain names into IP addresses. nslookup is a command-line tool to query DNS servers for:
  • IP addresses
  • Mail servers (MX)
  • Associated organizations (SOA)
  • Extra text data (TXT)
